Games

Tank Defender This game was my final project for the Web Game Programming class at Centennial College. You control a tank using your mouse. The tank is always shooting projectiles that can destroy the enemies or obstacles that are coming towards you. You can also run over the enemies with the tank, just don't hit the TNT. If you hit three TNT's the game is over. You also lose if twelve enemies make it past you.
